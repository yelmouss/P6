## How To Use

```bash

# Clone this repository or download

# Go into the repository
$ cd backend

# Install dependencies
$ npm install

# Run the app backend in http://localhost:3000
$ npm start


# Go into the repository
$ cd frontend

# Install dependencies
$ npm install

# Run the app
$ ng serve
U can use the app on the browser in http://localhost:4200

```
